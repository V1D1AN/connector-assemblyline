# Documentation Images

This directory contains screenshots and images for the documentation.

## Required Images

Please add the following screenshots:

1. **enrichment-note.png** - Screenshot of the enrichment note created by the connector
2. **malware-analysis.png** - Screenshot of the Malware Analysis section in OpenCTI
3. **relationships.png** - Screenshot of the relationships graph showing connections

## How to Capture Screenshots

1. Run the connector against a test file
2. Navigate to the analyzed Artifact in OpenCTI
3. Take screenshots of:
   - The "Notes" tab showing the AssemblyLine analysis note
   - The "Malware Analysis" section
   - The "Knowledge" tab showing relationships

## Image Guidelines

- Use PNG format
- Recommended width: 800-1200px
- Blur or redact any sensitive information
- Use consistent browser window size
